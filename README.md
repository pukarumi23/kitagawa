## 📲 Instalación y Ejecución en Termux

Si deseas ejecutar el bot en Termux (Android), sigue estos pasos sencillos:

1. **Actualiza paquetes básicos**
    ```sh
    pkg update && pkg upgrade
    ```
2. **Instala Git y Node.js**
    ```sh
    pkg install git nodejs
    ```
3. **Clona este repositorio**
    ```sh
    git clone https://github.com/pukarumi23/kitagawa.git
    ```
 4. **Ingresar a la carpeta proyecto**
    ```sh
    cd kitagawa
    ```
5. **Instala las dependencias**
    ```sh
    npm install
    ```
6. **Inicia el bot**
    ```sh
    npm start
    ```
7. **Descargar termux esta version recomendada**
    ```sh
    https://files.catbox.moe/b6z9ml.apk
    ```

¡Listo! El bot debería iniciar y mostrarte un código para vincularlo con WhatsApp Web.
